# Launch checklist (for the human — everything past this point needs you)

Everything that could be built without an account has been built and
tested. These are the only steps left, in the order that unblocks the most
value fastest:

1. **Create a GitHub repo** (~2 min) — unzip this, `git init`, push to a new
   public repo under whatever account/org you want it to live under. This
   alone lets people find, star, and fork it — the free distribution loop
   starts here, no payment step needed yet.
2. **Publish to PyPI** (~5 min, free) — `pip install aivisible-fix` by name
   instead of a zip download. Needs a PyPI account (free, email-verified).
3. **(Optional, once step 1 shows real usage) Payment processor** — only
   worth doing after the free tool has actual stars/downloads/usage
   validating demand for a paid tier (hosted continuous monitoring +
   auto-PR). Doing this before step 1 has traction is premature.

Nothing above requires touching any of your existing repos or businesses —
this is fully separate, per your instruction.
